# DebugAssist

**Augment your debugging workflow, don't automate it away.**

DebugAssist is a VS Code extension designed to help you understand *why* your code isn't working. Instead of just giving you a "fix" to copy-paste, DebugAssist provides contextual hints and analysis to help you grow as a developer.

Powerful, agentic AI tools like Cursor can abstract away the debugging process, limiting learning opportunities. Our tool offers a more balanced approach, providing targeted assistance to augment‚Äînot automate‚Äîa developer's understanding.
